0x02 Policy 参数解读

在上面的示例Policy中，有几个参数还是比较有价值的，一一解读一下ForAnyValue、IpAddress、aws:SourceIp：

  1. ForAnyValue，只要有一个值在符合要求，那就是符合要求
  2. IpAddress，IP地址，支持普通的地址格式、CIDR格式
  3. aws:SourceIp，请求的客户端IP
